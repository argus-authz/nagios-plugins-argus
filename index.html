Nagios plugins for Argus

General Information

These probes work exclusively for EMI Argus version 1.5.0 and onward. Probes are available for the Argus PAP, PDP and PEP Server.

Documentation

The complete documentation of the probes can be found online: https://twiki.cern.ch/twiki/bin/view/EGEE/ArgusEMINagiosProbes
